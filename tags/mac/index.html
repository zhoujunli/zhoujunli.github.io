  • mac下安装emmet后无法使用快捷输入的解决方法

    2017-06-25 |
    • { mac }
    • { emmet快捷键 }

    mac下安装emmet后无法使用快捷输入的原因可能是：PyV8安装失败

    解决方法：手动安装PyV8

    1. 在https://raw.github.com/emmetio/pyv8-binaries/master/pyv8-osx-p3.zip 下载需要的PyV8

    2. 在sublimeText上进入Browse Packages

    3. 上一步操作后会跳到packages文件夹内部，返回上一层文件夹，找到跟packages同级的install packages

    4. 进入install packages文件夹内，若里面没有PyV8文件夹，则新建一个，若有，就讲刚才下载的PyV8的压缩文件解压后，放到PyV8文件夹内。

    5. 重启sublimeText，输入html:5或者！后tab键，若成功出现html代码，则说明PyV8已经安装成功。
