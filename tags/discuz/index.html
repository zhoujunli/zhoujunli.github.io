  • 关于discuz论坛头像显示不出来的问题

    2015-02-01 |
    • { discuz }
    • { 论坛头像 }

    关于discuz论坛头像显示不出来的问题（如下图），

    且当你审查元素时，头像的路径是以localhost打头的话（又如下图)，

    且当你在控制器将localhost修改成你的网站网址时，头像显示正常的话（再如下图）

    那就要以下几点就是我们需要考虑修改的地方了

    1.论坛(bbs)的config文件夹中的配置文件–>config_global.php，config_global_default.php，config_ucenter.php，config_ucenter_defalut.php这四个文件。
    a.其中，config_global.php，config_global_default.php这两个配置只需将数据库名称及密码修改成跟你服务器数据库配置相同即可。

    b.对于config_ucenter.php，config_ucenter_defalut.php这连个配置文件除了要将数据名称及库密码修改成跟服务器数据库配置相同，config_ucenter.php需将
    define('UC_API', 'http://localhost/www/bbs/uc_server');
    修改为
    define('UC_API', 'http://www.gou39.com/www/bbs/uc_server');

    而对于config_ucenter_default.php，我们则需做出如下修改：

    2.改完以上文件，千万别以为这样就改完了。还有一个很容易被忽略的文件：即是你论坛下（bbs）的uc_server\data下的config.inc.php,你还需将其中的数据库名称和改成与你服务器相同的配置。
    做出以上修改之后，刷新页面，看看头像是否出现了。如果出现，修改就到此为止。倘若头像还出不来，那么请往下看：

    3.进入到你论坛的管理中心->Ucenter->应用管理->编辑，查看你当前论坛配置的应用IP是否为空，如果为空，请将你的服务器IP填上（尽管应用IP后面提示“正常情况下留空即可”，但是特殊情况特殊对待，在你做了以上两步的修改之后还无法显示头像，那就请你将IP地址填上），然后更新缓存

    4.改完IP地址之后，刷新页面，头像就出来，审查头像元素，
    头像的src也变成了http://gou39.com了

    PS:在这里仅仅是将我个人遇到的discuz论坛头像无法显示的问题作了一个思路整梳理以及总结。不代表你们遇到的问题都必须照着这个来修改。个人写的总结比较少。可能会出现比较多的纰漏。
